# Sync Watch

Real-time synchronized video watching platform for remote collaborative viewing with text chat and playback synchronization.

## Features

- Synchronized video playback across participants
- Real-time text chat alongside video
- Room creation with shareable invite links
- Play, pause, and seek sync across all viewers
- Support for multiple video sources

## Tech Stack

JavaScript, WebSocket, Node.js, Express, HTML

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/sync-watch.git
cd sync-watch
npm install
```

### Usage

```bash
npm start
```
